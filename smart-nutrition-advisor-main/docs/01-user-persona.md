# User Persona & User Journey

## 📌 Persona 1: Minh - Sinh viên đại học

| Thuộc tính | Chi tiết |
|------------|----------|
| **Tên** | Nguyễn Văn Minh |
| **Tuổi** | 21 tuổi |
| **Nghề nghiệp** | Sinh viên năm 3, ngành CNTT |
| **Mục tiêu sức khỏe** | Giảm cân (từ 78kg → 70kg) |
| **Pain Points** | Không biết nên ăn gì để giảm cân hiệu quả; thường ăn mì gói và fast food; không có thời gian tìm hiểu dinh dưỡng |
| **Nhu cầu** | Cần một gợi ý meal plan đơn giản, dễ thực hiện, phù hợp ngân sách sinh viên |
| **Tech Savvy** | Cao - sử dụng smartphone và laptop hàng ngày |
| **Thiết bị** | Laptop + Điện thoại |

### User Journey - Minh
```
1. Awareness: Minh nhận ra cần giảm cân sau kỳ nghỉ hè
2. Discovery: Tìm thấy Smart Nutrition Advisor qua bạn bè giới thiệu
3. Registration: Đăng ký tài khoản với email sinh viên
4. Input: Nhập thông tin (21 tuổi, 78kg, 170cm, mục tiêu: giảm cân)
5. Result: Nhận BMI = 27.0 (Overweight), Calories: 1800 kcal/ngày
6. Meal Plan: Xem meal plan 3 bữa phù hợp
7. Action: Áp dụng meal plan vào cuộc sống hàng ngày
8. Return: Quay lại cập nhật cân nặng sau 2 tuần
```

## 📌 Persona 2: Linh - Dân gym

| Thuộc tính | Chi tiết |
|------------|----------|
| **Tên** | Trần Thị Linh |
| **Tuổi** | 26 tuổi |
| **Nghề nghiệp** | Nhân viên văn phòng, tập gym 4-5 buổi/tuần |
| **Mục tiêu sức khỏe** | Tăng cân / tăng cơ (từ 52kg → 58kg) |
| **Pain Points** | Tập gym đều đặn nhưng không tăng cân được; không biết cần bao nhiêu calories/ngày; khó tính toán macro nutrients |
| **Nhu cầu** | Cần biết chính xác lượng calories cần nạp và meal plan giàu protein |
| **Tech Savvy** | Trung bình-Cao |
| **Thiết bị** | Điện thoại (chủ yếu) |

### User Journey - Linh
```
1. Awareness: Linh muốn tối ưu chế độ ăn để hỗ trợ tập gym
2. Discovery: Tìm thấy app trên mạng xã hội
3. Registration: Đăng ký nhanh bằng email
4. Input: Nhập thông tin (26 tuổi, 52kg, 162cm, mục tiêu: tăng cân)
5. Result: Nhận BMI = 19.8 (Normal), Calories: 2400 kcal/ngày
6. Meal Plan: Xem meal plan giàu protein, nhiều bữa phụ
7. Action: Kết hợp meal plan với lịch tập gym
8. Return: Theo dõi tiến trình hàng tuần
```

## 🔴 Pain Points chung
1. **Thiếu kiến thức dinh dưỡng** - Không biết nên ăn gì, bao nhiêu
2. **Quá nhiều thông tin** - Internet có quá nhiều lời khuyên mâu thuẫn
3. **Thiếu cá nhân hóa** - Các app hiện tại quá chung chung
4. **Phức tạp** - Các app dinh dưỡng hiện tại yêu cầu nhập quá nhiều thông tin
